# MQB MFS PQ25  Adapter
LIN MFSW Adapter — VW MQB Evo Steering Wheel on PQ25 Platform
(на основі проєкту mka1ugin/LIN_MFSW_Adapter — адаптер керма Skoda MQB для платформи PQ46)

Про проєкт
Адаптер LIN-шини для встановлення мультифункціонального керма VW MQB Evo (покоління 2020+, з кнопкою VIEW) на автомобілі платформи PQ25 (VW T5/T6 та інші на платформі PQ25). Побудований на STM32F103, вмикається в розрив LIN-шини між BCM автомобіля та кермом і виконує двосторонню трансляцію протоколів у реальному часі.
Можливості

✅ Мультимедійні кнопки (гучність, перемикання треків)

✅ Керування меню приладової панелі (▲/▼, OK, Next/Prev)

✅ Кнопки телефона та голосового керування

✅ Підсвітка керма з трансляцією яскравості від штатного BCM

✅ Емуляція діагностичних відповідей для BCM (кермо виглядає як штатне PQ)

🔧 У розробці (апаратна підтримка у ревізії 2 плати): круїз-контроль, клаксон, підрулюві пелюстки






About
A LIN bus adapter for retrofitting a VW MQB Evo multifunction steering wheel (2020+ generation, with VIEW button) onto PQ25 platform vehicles (VW T5/T6 and others PQ25). Built around an STM32F103, it sits in-line between the car's BCM and the wheel, performing real-time bidirectional protocol translation.
Features

✅ Media buttons (volume, track prev/next)

✅ Instrument cluster menu control (▲/▼, OK, Next/Prev)

✅ Phone and voice control buttons

✅ Wheel backlight with live brightness translated from the stock BCM

✅ Diagnostic response emulation (the wheel appears stock to the BCM)

🔧 In progress (hardware support in PCB rev. 2): cruise control, horn, shift paddles
